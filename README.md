### 2019-10-29
```
若教眼底无离恨，不信人间有白头

(如果不是眼下亲自感受这离愁别恨的苦楚，根本不会相信这世上真会有人伤心白头)

                                       -- 辛弃疾 《鹧鸪天·代人赋》
                             
```

### 2019-10-30

```
毋意，毋必，毋固，毋我。

（不凭空臆测，不绝对肯定，不固执己见，不自以为是）

                                       -- 孔子 《论语·子罕篇》
                                       
```
